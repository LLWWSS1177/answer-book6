# 答案书6

这是 “答案书6” 仓库的说明页。下面列出可用的页面和操作说明。

- 目录页（建议作为仓库主页或 GitHub Pages 首页）：[index.html](./index.html)
- 当前答案文件： [答案1 (answer1.html)](./answer1.html)

如何使用
1. 本地预览：在仓库根目录打开 `index.html` 或 `answer1.html`（双击或用浏览器打开）。
2. 启用 GitHub Pages（可选）：进入仓库 Settings -> Pages，选择 branch（例如 main）并保存。Pages 会把 `index.html` 作为站点首页（若使用默认设置）。
3. 添加新答案：在根目录新增 `answer2.html`、`answer3.html` 等，然后编辑 `index.html` 或本 README.md 添加链接。
